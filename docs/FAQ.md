# FAQ

## Does this remove shared dependencies?
No. Shared dependencies are kept automatically if another installed mod/map still needs them.

## What does savegame protection do?
It scans selected savegame XML files and keeps dependency mods that are still referenced there.

## What was fixed in v1.1.1?
The missing **Step 4 – Remove Files** section was restored so the new interface once again includes the final remove controls.
