FS25 Map Cleaner

Remove a selected FS25 map and only the dependency mods unique to that map.

Download Installer Download Portable ZIP

No Python needed. The installer lets the user choose the install drive/folder and optionally create a desktop icon.

How it works

  1. Pick your FS25 mods folder
  2. Scan installed mods
  3. Select the map
  4. Analyze dependencies
  5. Delete the selected map and only the unused dependency mods
